# 🌍 Travel Explorer

> Discover Indian beaches, ancient temples, and destinations across the world — with transparent INR pricing and smart search.

**Live demo:** [travel-recommendation-website.vercel.app](https://travel-recommendation-website.vercel.app/)

---

## ✨ Features

- 🔍 **Live fuzzy search** — find destinations even with partial or approximate queries (e.g. typing "Kerala" surfaces Varkala Beach)
- 🏷️ **Category filters** — India, Beaches, Temples, Countries
- 💰 **Dual currency support** — toggle between ₹ INR and $ USD pricing
- ↕️ **Sorting** — A–Z, Z–A, and Top Rated
- ❤️ **Favorites** — save destinations you're interested in
- 🧳 **Trip Plan builder** — add multiple destinations to a running trip plan and send it to a travel desk
- 📅 **Booking form** — collects name, email, destination, departure date, traveler count, and budget tier
- 📊 **Live stats** — destination count, category count, India picks, and 5-star picks
- 📱 **Responsive design** — works across mobile and desktop
- ✉️ **Newsletter signup** and a **contact form** for trip inquiries

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm / yarn / pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/<your-username>/<your-repo-name>.git
cd <your-repo-name>

# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the app.

### Build for production

```bash
npm run build
npm start
```
